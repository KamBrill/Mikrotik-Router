# Mikrotik RB5009 Home Lab Setup

# Overview
A detailed documentation of my journey configuring a MikroTik RB5009UG+S+IN for a secure, segmented home lab network. This project will include Firewall configuration, VLAN segmentation, and VPN setup.

# Objectives
- Replace current router with a more customizable router
- Implement VLAN segmentation (LAN, IoT)
- Configure a firewall using best practices for security
- Set up a Wireguard VPN for secure remote access
- Establish proper DNS and DHCP services
- Document the entire project for learning

# Hardware and Software
- **Hardware:** Mikrotik RB5009UG+S+IN
- **OS:** RouterOS
